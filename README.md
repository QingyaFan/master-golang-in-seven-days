# master-golang-in-seven-days

每个领域都有一个具有支配地位的语言，例如web开发方面是Javascript，nodejs本来是后端的技术，也归入了大前端，作为前端开发必备的技能。在容器、云计算领域，这个语言无疑是golang，kubernetes、docker都是golang实现，做devops的非常有必要学习golang。而且Golang原生支持并发，执行效率高、静态类型、无runtime依赖等，这些特性非常吸引人。于是有了这个项目。

与网上充斥的`一小时入门到精通`系列不同，我们保守和真实一点，不要急功近利，催熟的瓜不甜。七天，是一个里程碑，希望在七天的时间里，可以对golang有一个全面的认识，要想精通肯定要有实际项目的历练，祝好！
